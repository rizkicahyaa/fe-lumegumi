# Folder Gambar Carousel

Tambahkan gambar slide carousel Anda di folder ini dengan nama file berikut:

- `slide-1.jpg` → Slide pertama (Bean & Debt)
- `slide-2.jpg` → Slide kedua
- `slide-3.jpg` → Slide ketiga

## Tips
- Resolusi yang disarankan: **1920 × 700 px**
- Format yang didukung: `.jpg`, `.png`, `.webp`
- Setelah menambahkan gambar, edit array `slides` di file `app/components/HeroCarousel.tsx` untuk mengubah judul dan deskripsi setiap slide.
